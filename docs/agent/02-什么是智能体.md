### 发展历程
```
1. 符号智能体 (1950年代-1980年代)
依赖符号逻辑,使用逻辑规则和符号来封装知识并促进推理过程
缺点: 逻辑过于死板，依赖人为定义规则和符号,在处理不确定性和大规模现实世界问题时面临局限性
技术特点:
1.1：基于符号逻辑和规则推理(如专家系统)
1.2:核心组件:知识库,推理引擎,解释器
1.3：优点:逻辑透明,推理可解释
1.4:僵化的规则难以处理不确定性,复杂推理效率低

2. 反应式智能体（20世纪80年代-90 年代）
只关注智能体和环境之间的交互,强调快速和实时响应.基于感知-行动循环,有效的感知环境并做出反应,
需要较少的计算资源,能实现更快的响应，但可能缺乏复杂的高级决策和规划能力
技术特点:
  2.1：依赖感知-循环能力,强调实时响应(如机器人避障)
  2.2：设计模式: 基于条件规则(if-else)或行为分层(brooks的包容框架)
  2.3：优点: 低计算开销，快速响应
  2.4：局限: 缺乏长期规划能力

3. 基于强化学习的智能体（1990年代至2010年代）
如何让智能体通过与环境的交互进行学习,使其在特定的任务中获得最大的累计奖励.能够在位未知环境中
自主学习,无需人类的明确干预.如著名的阿尔法围棋
  3.1: 通过环境交互最大化累积奖励。
  3.2: 传统方法：Q-learning（1992）、SARSA（1994）。
  3.3: RL突破：深度神经网络+强化学习（RL），处理高维输入（如视觉）
  3.4: 优点：自主学习复杂策略。
  3.5: 局限：样本效率低、训练不稳定。

4. 具备迁移学习和元学习能力的智能体（2010年代末至2020年代初）
迁移学习来加快对新任务的学习速度,减轻了新任务的训练负担,在不同任务间共享和迁移,提高学习效率,
性能和泛华能力。元学习侧重于学习如何学习,引入后,使智能体能够从少量样本中快速推断出
针对新任务的最优策略。当源任务和目标任务之间存在明显差异时,迁移学习效果可能达不到预期,
并且可能出现负迁移
技术特点：
  ‌4.1：迁移学习‌：跨任务知识复用（如预训练模型微调）。
 ‌ ‌4.2：元学习‌：学习如何学习（如MAML，2017）。
  ‌4.3：优点：少样本学习、快速适应新任务。
  ‌4.4： 局限：任务差异导致负迁移，元学习预训练成本高。
 </br> 
5. 基于大语言模型的智能体（2020年代初至今）
 将大语言模型作为智能体的大脑或控制器,并通过多模态感知和工具使用等策略来扩展器感知和行动空间。
 这些基于大语言模型的智能体能够通过思维链（CoT）和问题分解等技术，展现出可与符号智能体相媲
 美的推理和规划能力。它们还能通过从反馈中学习并执行新动作，获得与环境交互的能力，这与反
 应式智能体类似。同样，大语言模型在大规模语料库上进行预训练，并表现出少样本和零样本泛化能力，
 能够在无需更新参数的情况下在不同任务之间实现无缝迁移。基于大语言模型的智能体已被应用于
 各种现实场景中，比如软件开发 和科学研究。由于它们具备自然语言理解和生成能力，
 这些智能体能够彼此无缝交互，从而引发多个智能体之间的协作与竞争。此外，研究表明，
 允许多个智能体共存可能会导致社会现象的出现 。
 技术特点:
   5.1: 以LLM为核心控制器（如GPT-4、PaLM等）。
   5.2: 能力扩展：多模态感知、工具调用（如ChatGPT插件）、多智能体协作。
   5.3: 优点：零样本推理、自然语言交互、社会行为模拟。
   5.4: 局限：幻觉问题、实时行动能力受限。
```

### 什么是AI  Agent
```
是一种能够感知环境,自主进行决策并利用工具采取行动,以达成目标的智能实体
具体如下的特征:
1. 自主性(Autonomy):能够独立执行任务,无需人工干预
2. 感知(Preception): 通过各种传染器(如摄像头和麦克风), 感知和解释它们所处的环境
3. 反应(Reactivity): 对环境中的及时变化和刺激快速做出反应的能力,意味着智能体能快速感知周围
环境并做出反应
4.推理和决策(reasonsing and decision-making)：分析数据,推理,规划并做出决策,决定下一步方案
5.主动性(Pro-activeness):围绕特定目标主动采取行动,能在与环境的交互过程中不断学习,调整行动
策略,以更好的实现目标
6. 学习力(Learning):随着时间的推移和数据的积累,AI agent能够不断的学习,
持续优化自己的行为和决策能力.
7.社交能力(social ability):智能体通过某种通信语言和其他智能体或人类进行交互的能力,
例如理解和响应自然语言,识别语音以及通过文本交换信息
```

### AI Agent 和 大语言模型(LLM)的区别
```
大语言模型如人类的大脑可以进行思考分析,推理决策,但没有感官(视觉,听觉,味觉等).四肢(手和脚)和工具.
无法观察环境和世界互动
AI agent相当于给大语言模型安装了感官和四肢,可以感知周围环境,并使用工具与环境交互,影响环境,
扩展了大语言的感知空间和行动空间,而且还有记忆能力,能够对过去的行为进行反思,累计经验,不断迭代
学习以优化未来的规划和决策。
举个例子:  问 '我想去北京旅行,有什么建议'
LLM: 列出一些著名景点,美食,推荐酒店. 你得接着问 酒店怎么到景点 门票怎么卖 你得每次都问
AI agent:他会根据你提供的预算,时间等因素,给你列出具体的行程安排.第一天去哪里,怎么坐车。
买票,查询好路线，在哪里吃什么,类似与你的私人助理.他还可以记住你的爱好,如喜欢安静,他会
推荐安静的住宿。思考一系列复杂的任务。
```
