## 核心概念
```
由三个关键部分组成: 大脑,感知,行动
1. 感知(Perception)
指 AI Agent 观察周围环境，并从环境中收集信息，提取有用的数据。Agent 所收集的信息包括文本、
听觉、视觉等多模态领域。为了使智能体更好地理解环境，能够接收多模态的信息是至关重要的。

2. 大脑(Planning/decision)
大脑模块核心是进行规划和决策，它根据感知模块收集的环境状态信息，
结合内部的世界知识和记忆进行规划并做出决策，为达成目标任务生成下一步的行动计划。
规划和决策通常由LLM（大语言模型）来承担，除此之外，还包括：
    2.1: World Knowledge：具备丰富的世界知识，由大语言模型在训练中得到并固化在模型参数中。
    2.2: Memory：记忆保持了历史的思考决策过程及过去的行动计划。
    2.3: Reflection：反思是根据当前的环境状态信息和历史行动结果进行总结，生成更高级别
    抽象的见解，作为经验信息，并存储在记忆中，以优化未来的决策，该功能使得AI Agent具有
    自我迭代学习能力，如：在错误中总结教训并在后续步骤中完善。

3. 行动(Action)
指 AI Agent 执行策略做出的具体行动，比如输出文本，使用搜索工具查询网页信息、
调用API访问外部服务等等。行动会产生多样的结果，并影响环境状态。
行动结果和环境状态的变化进一步被感知模块观察到，作为下一轮的输入。
这个过程不断迭代，直到完成目标。

```

### 大脑模块
```
1. 自然语言交互
2. 知识
3. 记忆
4. 推理与规划（计划制定 计划规定)
5. 迁移和泛化能力
```

### 感知模块
```
1. 文本输入
2. 图像输入
3. 视频输入
4. 听觉输入
5. 其他输入
```

### 行动模块
```
1. 文本输出
2. 工具使用
3. 量身行动
```


## 核心流程
```
AI Agent=Perception（Observation）+ Brain（LLM+State+Memory+Reflection）+ Action（Tool）
```
